# conceptin_UML_FAI
Ce projet nous a permis de modéliser de façon complète le fonctionnement d'un système ISP à travers 6 diagrammes UML complémentaires — couvrant aussi bien la structure statique que les comportements dynamiques. 6 diagrammes UML couvrant les cas d'utilisation, les classes, les séquences, les activités et les états du système.
